*회원가입
 나이, 펫수는 "숫자"로 입력해야함.

*펫 등록
 펫 나이, 가격은 "숫자"로 입력해야함.


1. 로그인 후 나의 게시글의 후보자 중 선택하여 매칭해보자.
[로그인]
ID: owner01 / PW: aaa 으로 로그인 -> [메뉴]나의 게시글 보기-> #1'별이 분양합니다.' 글 -> 신청자 보러가기 -> person01 클릭 -> "매칭 완료"

2. 나의 매칭 목록에서 사후관리 게시판에 입장해보자. (계속 owner01에 로그인한 상태)
펫이'달이'인 사후관리 입장 클릭 -> 사후관리 게시판 (게시글 작성, 수정, 삭제 가능)
-> person01이 작성한 글 제목 클릭 -> 삭제 클릭 -> "본인이 작성한 글만 삭제할 수 있습니다." 경고창 뜸.(게시글 수정도 마찬가지)

3. 매칭 실패한 경우를 살펴보자.
ID: person02 / PW: aaa 로그인 -> [메뉴]내가 신청한 목록 보기 -> 방금 매칭 완료한 '별이'에 대한 글이 매칭 "실패" 인걸 확인할 수 있다.

4. 신청한 리스트 중 하나를 삭제해보자. (계속 person02에 로그인한 상태)
[메뉴]내가 신청한 목록 보기 -> '귀여운 나비분양!' 게시글 삭제 클릭 -> "삭제하시겠습니까?"경고창 뜸 -> 확인 클릭 시 삭제 됨을 알 수 있다.


5. 비즈니스로직 - 사용자의 기호를 분석하여 게시글을 추천해준다.
ID: person01 / PW: aaa 으로 로그인 -> '말티즈 감자 분양' 작성자 owner03의 게시글을 신청 
-> [메뉴]내가 신청한 목록 보기 클릭 -> '감자(말티즈)-대기중', '별이(말티즈)-대기중', '달이(불독)-성공', '나비(페르시안)-대기중', '티티(먼치킨)-대기중' 상태임. 따라서 "말티즈"에 대한 게시글을 많이 신청함.
-> [메뉴]추천 게시글 클릭 -> "말티즈"에 대한 모든 게시글을 추천해주는 것을 확인할 수 있다.


*모든 게시글의 수정 및 삭제는 작성자와 현재 로그인한 사람이 동일해야 가능하다.
*이미 매칭 완료된 게시글은 신청 불가능하다.
